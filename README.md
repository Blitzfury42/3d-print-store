# 3D Print Store

Welcome to the 3D Print Store project! This repository contains a simple online platform for ordering 3D prints from various models.

## Setup Instructions

Follow these steps to get your development environment set up:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Blitzfury42/3d-print-store.git
   cd 3d-print-store
   ```

2. **Install Dependencies**
   Make sure you have Node.js installed. Then run:
   ```bash
   npm install
   ```

3. **Start the Server**
   To start your local development server, run:
   ```bash
   npm start
   ```

4. **Visit the Application**
   Open your web browser and go to `http://localhost:3000` to see the application in action.

## Features
- User authentication
- Model uploading
- Order management
